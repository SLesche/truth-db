Welcome to Our Website

Our database is a broad collection of attentional control data, freely available for researchers, educators, and enthusiasts to explore and utilize. This open database aims to support scientific discovery and foster collaboration across disciplines.

We are continuously expanding our database and welcome contributions from the community. If you have attentional control data to share, please visit our Contribute Data page for more details on how to submit your work.

Working with the Database

To help you explore and work with the data, we provide several tools and resources:

  • Explore the Database: Use our interactive Explore page to browse, search, and visualize the dataset.
  • R Package: For advanced analysis, download our dedicated R package that simplifies interaction with the database. This package allows you to easily query, analyze, and visualize the data directly in R.

We are committed to making this database as accessible and useful as possible. If you have questions or suggestions, feel free to contact us.
